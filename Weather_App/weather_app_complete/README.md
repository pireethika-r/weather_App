
# Weather App

## Features
- Search weather by city
- Real-time weather data
- Temperature, humidity, wind speed
- Weather icons
- Error handling
- Responsive design
- Backend API integration
- Firebase configuration

## Technologies
- HTML
- CSS
- JavaScript
- Node.js
- Express.js
- OpenWeatherMap API
- Firebase

## Run Frontend
Open:
frontend/index.html

## Run Backend

cd backend
npm install
npm start

Server:
http://localhost:5000

## OpenWeatherMap API
Replace:
YOUR_OPENWEATHER_API_KEY

inside backend/server.js

## Firebase
Configure Firebase inside:
database/firebase-config.js
